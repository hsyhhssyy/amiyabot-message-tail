# 兔兔小尾巴

开启该功能以后，在控制台的配置项中打开启动开关。该插件就会在兔兔回复的每一句话后面都加上自定义的一句话，如下图所示。

每次兔兔启动后，都会自动将该启动开关切回到关闭状态。
